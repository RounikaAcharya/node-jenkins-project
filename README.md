# Node.js Jenkins CI/CD Project

This project demonstrates a basic Continuous Integration (CI) pipeline using Jenkins.

 Tech Stack
- Node.js
- Express.js
- Jenkins
- Git & GitHub

 Features
- Simple Node.js web server
- Version control using GitHub
- Jenkins integration for automated build
- Executes npm install and runs the app automatically

 Workflow
1. Code is pushed to GitHub
2. Jenkins pulls the latest code
3. Installs dependencies using npm
4. Runs the application

Output
Displays: "Hello from Jenkins CI/CD 🚀" on localhost

Purpose
To understand the basics of CI/CD pipeline setup using Jenkins.
